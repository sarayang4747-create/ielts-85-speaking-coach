# Question Bank Workflow

This skill does not include a current-season IELTS question bank. The learner should upload or paste a legally obtained question bank PDF, document, or text.

## When A User Uploads A Question Bank

1. Confirm what file/content was provided.
2. Extract or read the question list if the platform supports it.
3. Classify questions into Part 1, Part 2, and Part 3.
4. For Part 1, classify common question types where possible.
5. For Part 2, map each cue card to 人物 / 地点 / 事件 / 物品 or "needs custom story".
6. For Part 3, map questions to the eight topic areas and likely thinking angles.

## If File Reading Is Not Available

Ask the learner to paste:

- a few questions they want to practice; or
- one topic section at a time; or
- screenshots converted to text by their device.

Keep the workaround simple. Do not make the learner do heavy formatting.

## Practice Modes

### Quick Drill

Pick 3-5 questions from one Part. Give immediate feedback after each answer.

### Framework Drill

Use one framework repeatedly until the learner can answer smoothly.

### Weakness Drill

Choose questions that target the learner's main issue, such as weak detail, unclear opinion, or Part 3 abstraction.

### Topic Cluster

For Part 2/3, group related questions so the learner can reuse one story or one viewpoint across multiple questions.

## Output Style

When planning practice, keep it light:

- Today we will practice 3 questions.
- Main focus: clear opinion + one concrete detail.
- After each answer, I will only point out the top 1-2 fixes.

Avoid long study plans unless the user asks.
