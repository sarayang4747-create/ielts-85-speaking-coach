# Example Prompts

下面这些提示词可以直接复制给支持 Skill 的 agent 使用。

## 1. 新手开始

```text
我的目标口语分数是 6.5。我不确定自己现在最该练什么，请用 IELTS 8.5 Speaking Coach by Sara 的方法带我开始。
```

## 2. 先诊断

```text
我想先做一个口语诊断。请你不要只给分数，重点帮我判断我最卡分的问题是什么，然后告诉我下一步该练什么。
```

## 3. 学 Part 1 框架

```text
我想练 Part 1。请先用一个最小框架教我怎么答，然后马上给我一道题练习。不要一次讲太多。
```

## 4. 拆一道 Part 1 题

```text
Part 1 题目：Do you like reading?

我不知道怎么展开。请先帮我激活自己的经历，不要直接给我完整答案。
```

## 5. 练 Part 2

```text
Part 2 题目：Describe a person who helped you.

请用 Sara 的四类素材逻辑帮我拆题，先给我一个回答骨架，不要给完整背诵稿。
```

## 6. Part 2 目标 7+

```text
我的目标口语是 7+。这道 Part 2 我不想用太通用的素材，请帮我挖一个更个人化的核心故事，再教我怎么迁移到类似题目。
```

## 7. 练 Part 3 ORECS

```text
Part 3 题目：Do you think children should learn how to plan ahead?

请用 ORECS 带我拆题：先帮我找观点，再让我自己答一版。
```

## 8. 改回答：第一轮

```text
题目：Do you enjoy your current age?

我的回答：
Yes, I like my age because I am free and I can do many things. I can study, work and meet friends. So I think it is good.

请按 Sara 的方法帮我看这段回答。先不要给完整高分改写，只指出最值得改的 1-2 个问题，给我几个关键表达，然后让我自己再改一版。
```

## 9. 改回答：第二轮

```text
这是我修改后的版本：
[粘贴你的第二版回答]

请继续帮我看，并给一个参考版。请说明这个参考版是示范如何展开，不是让我背。
```

## 10. 使用题库

```text
我上传了一份当季雅思口语题库。请先帮我把题目分成 Part 1 / Part 2 / Part 3，再给我一个轻量练习顺序。不要做很复杂的长期规划。
```

## 11. Part 3 没思路

```text
Part 3 题目：Why do people like using social media?

我没有观点。请先用 7 个万能切入点帮我找 2-3 个容易说的角度，然后让我选一个。
```

## 12. 要求粗略分数区间

```text
这是我的口语回答文字稿：
[粘贴回答]

请给我一个非常粗略的分数区间判断，但重点还是告诉我：最影响分数的问题是什么，以及我下一轮应该怎么改。
```
