# IELTS 8.5 Speaking Coach by Sara

一个不背模板的雅思口语方法论 Skill。

它不是一个实时语音陪练工具，也不是“高分答案生成器”。它更像一个随时可调用的雅思口语方法论教练，帮你用 Sara 的思路练习：

- 观点生成
- 细节展开
- Part 1 / Part 2 / Part 3 答题框架
- 题库拆解
- 回答复盘
- 个性化答疑

## 适合谁

尤其适合：

- 当前口语大概 5.0-5.5，想冲 6.0/6.5 的考生
- 不知道 Part 1 / Part 2 / Part 3 分别该怎么准备的人
- 一开口就没思路、答案太短、细节展开不出来的人
- 不想死背模板，但希望有一套稳定方法的人
- 想更系统地使用 AI 辅助雅思口语备考的人

如果你的目标是 7+，也可以使用这个 Skill，但你会更需要个性化素材、Part 3 抽象思维和表达精度训练。

## 它能做什么

你可以让它帮你：

1. **先诊断**：我不知道自己口语卡在哪里。
2. **学框架**：Part 1 / Part 2 / Part 3 到底怎么答。
3. **拆题目**：这道题我完全没思路。
4. **改回答**：帮我看这段回答哪里可以更好。
5. **用题库**：根据我上传的当季题库练习。

## 它不能做什么

这个 Skill 主要基于文字回答或语音转写稿来分析，不承诺：

- 实时听你说话
- 直接判断发音、语调和真实流利度
- 给出官方雅思评分
- 保证某个目标分数
- 内置当季雅思题库

如果你的 AI 工具支持语音输入，你可以先用语音回答，让工具转成文字，再让这个 Skill 根据文字稿复盘。

## 为什么不是普通 Chatbot

很多 AI 陪练每次都像重新开始：你要不断解释目标、标准和练习方式。

这个 Skill 的重点是把 Sara 的方法论沉淀进去：

- 不背完整模板
- 观点先行
- 用简单词把话说清楚
- Part 1 / Part 2 / Part 3 都有抓手
- 每次反馈都尽量告诉你下一步怎么练

## 怎么开始

### 如果你的 agent 支持 Skills

把这个项目作为一个完整 Skill 文件夹导入或安装。确保 `SKILL.md`、`references/`、`examples/` 都在同一个项目目录下。

安装后，可以直接这样问：

```text
我的目标口语分数是 6.5。我现在不知道自己主要卡在哪里，请先帮我做一个诊断。
```

或者：

```text
我想练 Part 1。请用 Sara 的方法带我从一个最小框架开始练。
```

也可以粘贴你的回答：

```text
这是我对 Part 3 题目的回答，请不要直接给完整改写，先帮我指出最值得改的 1-2 个问题，并让我自己再改一版。
```

更多示例见 [examples/prompts.md](examples/prompts.md)。

### 如果你的工具暂时不支持 Skills

也可以先复制 [examples/prompts.md](examples/prompts.md) 里的提示词使用。效果会弱一些，因为 agent 不一定能持续读取完整方法论，但仍然可以体验核心流程。

## 当季题库怎么用

本项目不内置雅思题库。你可以上传或粘贴自己合法获得的当季题库，之后让 Skill 帮你：

- 分类 Part 1 / Part 2 / Part 3
- 把 Part 2 题目映射到人物 / 地点 / 事件 / 物品素材
- 把 Part 3 题目归类到常见话题
- 选择练习顺序
- 分析你的回答
